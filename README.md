**Titanic Data Analysis**

**Project Overview**
This project performs data analysis on the Titanic dataset using Python and Jupyter Notebook. The goal is to explore relationships between different features and extract meaningful insights.

**Tasks Performed**
Data Cleaning
Handling Missing Values
Exploratory Data Analysis (EDA)
Correlation Heatmap
Data Visualization

**Key Insights**
Passenger class (Pclass) is strongly related to fare
People traveling with family are less likely to be alone
Adult males are more likely to travel alone

**Tools & Libraries**
Python
Pandas
NumPy
Matplotlib
Seaborn

**📂 Project Structure**
titanic-analysis.ipynb → Main notebook with code and analysis

**How to Run**
Download the notebook
Open in Jupyter Notebook
Run all cells

**Conclusion**
The analysis shows that factors like passenger class, fare, and family size have strong relationships and can be useful for further prediction models.
